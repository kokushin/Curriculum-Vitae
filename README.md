## 自己紹介

名古屋のWeb系専門学校を卒業後、新卒で株式会社アンティー・ファクトリーの名古屋支社へ入社。  
4年間、大手企業サイトのコーディングやWebアプリ開発などフロントエンド全般に従事する。

2018年4月、転職のため上京し、創業期の株式会社GIGへ入社。  
テックリードとして、「[Workship](https://goworkship.com/)」(フリーランス向けマッチングサービス)のフロントエンド全般を担当する。

2019年にフリーランスとして独立し、引き続きGIGにて新規事業「[LeadGrid](https://goleadgrid.com/)」(CMSサービス)の初期メンバーとして企画/開発に参加。

2020年に法人化し、株式会社メイクリーを創業。代表取締役に就任する。  
同年3月より自社の新規事業「[Makelearn](https://makelearn.net/)」（エンジニア向け学習サービス）がスタート。

好きなプログラミング言語はJavaScriptとCSS。

[社内インタビュー記事](https://giginc.co.jp/blog/interview/2795)
